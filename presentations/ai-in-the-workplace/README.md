# AI in the Workplace

## Title

Kuenstliche Intelligenz im Arbeitsalltag: Moeglichkeiten, Grenzen und verantwortliche Nutzung

## Audience

- non-specialists in company environments
- technically interested employees without deep AI background
- project managers, team leads, and business-side multipliers

## Learning Goal

The deck is designed to explain practical AI usage in everyday work, clarify realistic strengths and limits, and frame responsible usage with method and guardrails.

## Files

- `ai-in-the-workplace.pptx`: exportable presentation deck
- `build_presentation.py`: local generator for the PPTX file

## Notes

- The deck uses the trust-audited wording from the repository docs.
- It is a public showcase artifact, not a full internal training package.
- A PDF export is not included because no local office/PDF conversion tool is currently available.
