# AI Enablement Showcase

This repository is a curated public-facing showcase for AI enablement and education materials.

It is designed to present a small set of teaching artifacts such as slide decks, handouts, speaker notes, and supporting documentation in a way that is readable for recruiters, hiring managers, and technical stakeholders.

The current repository already includes the first content foundation for a public presentation project on responsible AI use in everyday work.

## Current Status

The accessible source paths reviewed for the initial discovery milestone did not contain clearly suitable public AI training artifacts that could be copied as-is.

What was found instead:

- software project documentation
- deployment and milestone notes
- one bot command reference PDF

Because none of those artifacts credibly demonstrate AI enablement or education on their own, they were not copied into this showcase.

Instead, this repository now contains:

- a public-facing repository shell for AI enablement materials
- a researched presentation brief for an AI enablement showcase deck
- a trust audit for the planned presentation claims
- a slide-by-slide script for the first presentation draft

## Why This Repository Exists

The purpose of this repository is to make AI enablement work legible as a professional artifact.

That means the repository should show:

- the ability to explain AI clearly to non-specialists
- practical guidance for workplace AI usage
- responsible framing of limitations, risks, and guardrails
- structured educational material rather than ad hoc notes

## Inclusion Criteria

Artifacts should meet all of the following before inclusion:

- clearly related to AI, MCP, prompt engineering, AI governance, or technical AI education
- understandable without private company context
- free of participant data, internal references, and confidential details
- usable as evidence of teaching, enablement, or technical explanation skill

## What Is Needed For V1

To complete a strong first public version, provide one to three real artifacts such as:

- a PDF or slide deck on an AI topic
- speaker notes for a technical AI presentation
- a short handout used in an AI workshop or enablement session

Once those are available, this repository can be turned into a compact public showcase with per-artifact context pages and clean navigation.

## Scope Note

This is a public showcase repository, not a private archive and not a dumping ground for raw workshop material. Only curated, public-suitable artifacts should be added.
